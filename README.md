# Quill - Journal Landing Page

A responsive landing page for a digital journaling application built while learning modern HTML and CSS.

## Overview

Quill is a frontend project focused on creating a clean and responsive user interface.  
The project helped me practice webpage structure, styling techniques, and responsive layouts.

## Features

- Responsive design for different screen sizes
- Modern landing page layout
- Navigation section
- Hero section
- Feature showcase section
- Call-to-action section
- Login and register pages

## Built With

- HTML5
- CSS3

## Concepts Practiced

- Semantic HTML
- CSS variables
- Flexbox
- CSS Grid
- Media queries
- Reusable CSS components
- Responsive units
